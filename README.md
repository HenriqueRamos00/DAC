# DAC
Trabalho de DAC TADS UFPR

## Frontend
```bash
cd frontend/
```

Para baixar as dependências (node_modules) use:
```bash
npm i
```

Para iniciar o frontend use:
```bash
npm run dev
```
Acesse em:
```bash
http://localhost:5173/
```
